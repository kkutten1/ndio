ndio.remote.OCPMeta module¶

class ndio.remote.OCPMeta.OCPMeta(hostname='lims.neurodata.io', protocol='http')¶

Bases: ndio.remote.Remote.Remote

OCPMeta Remotes enable access to the Metadata-OCP API endpoints (which can be found at api.neurodata.io/metadata/ocp/). This class is useful for reading metadata for existing projects, or, if you specify a key, adding new metadata, manipulating your existing metadata, and archiving old metadata. Archived metadata is still stored by the LIMS system.
